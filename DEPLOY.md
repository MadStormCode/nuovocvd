# Centro Vela Dervio - Deploy

## Build
```bash
npm run build
```

## Local Dev
```bash
npm run dev
```

## Database
Migrazioni applicate automaticamente via Supabase MCP.

## Env
- `NEXT_PUBLIC_SUPABASE_URL`
- `NEXT_PUBLIC_SUPABASE_ANON_KEY`
- `SUPABASE_SERVICE_ROLE_KEY`
- `SUPABASE_DB_URL`

## Features
- Next.js 13 App Router
- Tailwind CSS + shadcn/ui
- Supabase Auth + PostgreSQL
- RLS policies
- Admin panel completo
- Multilingua (IT/EN)
